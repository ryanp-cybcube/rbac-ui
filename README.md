# rbac-ui

```
Do ‘create-react-app rbac-ui-test’ to create React project folder.
Import rbac-ui repo’s /common, /components, /containers, /existing folders into rbac-ui-test /src folder.
Update rbac-ui-test App.js file with repo’s App.js file.
Update rbac-ui-test public/index.html with repo's index.html
Run and make sure dependencies are install with npm.
```
